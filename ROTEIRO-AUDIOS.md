## Dica para o ElevenLabs

No ElevenLabs, você pode usar a funcionalidade **"Projects"** para gerar todos os áudios de uma vez:
1. Crie um projeto novo
2. Cole cada texto como um bloco separado
3. Selecione a mesma voz para todos
4. Gere e baixe os arquivos
5. Renomeie conforme a estrutura acima

Ou use a **API** para automatizar:
```bash
curl -X POST "https://api.elevenlabs.io/v1/text-to-speech/VOICE_ID" \
  -H "xi-api-key: SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text": "TEXTO_AQUI", "model_id": "eleven_multilingual_v2"}' \
  --output arquivo.mp3
```
